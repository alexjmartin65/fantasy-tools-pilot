🔧 How to Resolve

Option 1: Upgrade Node.js (Recommended)

  1. Visit nodejs.org
  2. Download Node.js v18.x or v20.x LTS
  3. Install and restart your terminal
  4. Run: npm install
  5. Run: npm run dev

Option 2: Use Compatible Versions

I've downgraded the build tools to be compatible with Node.js v14:

npm install npm run build npm run serve

📊 Technical Details

  • Framework: React 18 with TypeScript
  • Build Tool: Vite (downgraded to v3.2.7 for compatibility)
  • Styling: Tailwind CSS
  • State Management: React Context + useReducer
  • CSV Processing: Papa Parse library
  • Icons: Lucide React
